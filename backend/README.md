## Descrição

Backend - API de produtos com endpoints create - edit - update - delete

[API Doc](http://localhost:3000/api) Documentação simples da API - Swagger >> http://localhost:3000/api

## Instalação

```bash
$ npm install
```

Editar .env

```bash
$ cp .env.exemplo .env
```

## Rodar e Compilar o projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
